IDEA 之 常用快捷键😮

Posted on 2021-09-23，2 min read

🚩 一些常用的idea快捷键，方便查阅，随时更新。

Ctrl + Shift + A ：全局搜索设置

Shift + Shift ：全局搜索

Ctrl + Shift + 向右（左）箭头 ：选择

Alt + Shift + 向上（下）箭头：上或下整行选择

Ctrl + 斜杠 ：注释代码

Ctrl + Shift + 斜杠 ：写注释

Alt + 向下箭头 ：移动代码

Ctrl + NumPad - ：折叠代码

Ctrl + NumPad + ：展开代码

Ctrl + NumPad * ：展开所有代码

Ctrl + Shift + NumPad / ：折叠所有代码

Alt + Shift + Z ：增加环绕代码（for | if | try/catch）

Alt + R ：取消环绕代码

Alt + Y ：选择相同名字的代码（单个）

Alt + Shift + ：取消相同名字的代码（单个）

Ctrl + Shift + Y ：选择所有相同名字的代码

Ctrl + Shift + Enter ：添加括号（;）结束代码

Alt + Shift + R ：替换所有用到的变量

Alt + Shift + L ：添加该行语句的变量

Alt + Shift + M ：提取代码块，封装成一个方法

Ctrl + Alt + C ：将该字符变为公共变量

Ctrl + Alt + P ：将表达式提取为参数

Ctrl + Alt + L ：格式化代码

Alt + Shift + O ：高亮选择的所有使用变量

F3 ：跳转至类或接口的声明

Ctrl + T ：跳转至该类或接口的实现

F12 ：跳转至源代码

Ctrl + F3 ：查看该类的代码结构

Ctrl + K ：查找该变量被使用的下一个

Ctrl + Shift + K ：查找该变量被使用的上一个
