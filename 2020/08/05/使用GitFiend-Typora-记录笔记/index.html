使用GitFiend + Typora 记录笔记

本地已经拉取了gitee的一个仓库

  • 拉取gitee仓库

  • 运行GitFiend，打开本地拉取的仓库

  • 不能pull

解决

  • 用git生成ssh公钥

  • 在gitee中添加ssh公钥

  • ssh -T git@gitee.com 测试是否成功即可

自定义图床

配置Picgo

  • 安装Picgo

    $ npm install picgo -g

  • 查看路径

    $ which picgo

    注意：填入自定义命令时，路径需要将斜杠换一下

    如：C:\Users\XYSM\AppData\Roaming\npm\picgo u

    格式：路径 + 空格 + u

  • 安装插件

    $ picgo install smms-user

  • 进入C:\Users\XYSM

    XYSM为我的用户名，然后找到.picgo文件夹，打开config.json配置文件

  • 配置文件

  • 测试，点击图中3即可测试。这样每次复制图片后就会自动上传到你自己的smms中啦。
