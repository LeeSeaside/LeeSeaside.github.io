使用hexo+gihub搭建博客

安装所需环境

  • 安装Node.js

    下载地址：Nodejs中文网
    检查：Win+R 输入cmd进入，然后输入以下命令，出现版本号即安装成功。

  • 安装Git

    下载地址：Git
    检查：git --version

  • 安装Hexo

    新建一个文件夹，用于存放我们的博客文件。如：D:\learn\LeeSeaside。

    在自己的文件夹下右键Git Bash Here进入Git控制台窗口。

    输入npm i hexo-cli -g—> 安装Hexo。

    hexo -v—> 验证是否安装成功。

    hexo init—> 初始化文件夹。

    npm install—> 安装必备组件。

    hexo g—> 生成静态网页。

    hexo s—> 启动本地服务，然后输入localhost:4000查看效果。

    Ctrl+C—> 关闭服务。

注册Github账号

网址：Github

进去注册一个账号

  • 新建一个仓库

    仓库名：你的用户名 + github.io。

    注意：必须严格按照这种方式创建。

    比如我的则是：LeeSeaside.github.io。

    勾选以下选项，然后创建即可。

  • 设置

    点击Seetngs，找到GitHub Pages，点击Choose a theme，选择一个主题，保存即可。

    访问leeseaside.github.io即可看到效果。

配置SSH Key

生成SSH Key 后，前往Github点击头像下的setting，在SSH and GPG keys添加SSH key即可。

将获取到的信息复制到Key中，完成add操作。

检查：输入ssh -T git@github.com，如果出现表示成功。

配置Hexo

打开你创建的博客目录下的_config.yml文件，repository为你仓库地址。

安装插件：npm install hexo-deployer-git --save
